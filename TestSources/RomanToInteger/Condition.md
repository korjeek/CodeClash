 # Из римских в арабские!


На вход вы получаете строку, которая является записью римского числа ***str***.
>| Римская цифра | Арабская цифра |
|:---------------:|:---------------:|
| I | 1 |
| V | 5 |
| X | 10 |
| L | 50 |
| C | 100 |
| D | 500 |
| M | 1000 |


Верните число в арабской записи, учитывайте, что IV - это 4, IX - 9. Запись числа римскими цирами всегда корректна.


* Пример №1
> **Input**: str = "III"
> **Output**: 3

* Пример №2
> **Input**: str = "LVIII"
> **Output**: 58
<u>Комментарий</u> : L = 50, V = 5, III = 3 : 50 + 5 + 3 = 58 


* Пример №3
> **Input**: str = "MCMXCIV"
> **Output**: 1994
<u>Комментарий</u> : M = 1000, CM = 900, XC = 90, IV = 4 : 1000 + 900 + 90 + 4 = 1994
